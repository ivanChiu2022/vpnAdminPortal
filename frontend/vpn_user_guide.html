User Setup Guide

Follow the steps below to install WireGuard, open your VPN configuration file, and connect securely on your phone or laptop.

1

Download the WireGuard App

Install the official WireGuard application on your device first.

Mobile (iPhone / Android)
  • Open the App Store or Google Play
  • Search for WireGuard
  • Install the official WireGuard app
Laptop (Windows / macOS)
  • Open your browser
  • Go to the official WireGuard install page
  • Download the correct version for your device
  • Complete the installation
Open WireGuard Download Page
2

Download Your .conf File from Email

Your VPN email includes an attachment that contains your personal VPN configuration.

  1. Open the email from your VPN administrator
  2. Find the attached file ending with .conf
  3. Download the file to your phone or laptop
  4. Remember where the file is saved
Suggested Save Locations
  • Phone: Files app or Downloads folder
  • Laptop: Desktop or Downloads folder

Example filename: ivan_phone.conf or ivan_laptop.conf

3

Open WireGuard and Connect

Import your .conf file into WireGuard, then activate the VPN tunnel.

On Phone
  1. Open WireGuard
  2. Tap Add Tunnel
  3. Select Import from file
  4. Choose your downloaded .conf file
  5. Tap Activate
On Laptop
  1. Open WireGuard
  2. Click Add Tunnel
  3. Select Import tunnel(s) from file
  4. Choose your downloaded .conf file
  5. Click Activate

How to Confirm It Works

WireGuard shows the tunnel as Active
Your internet still works normally
Your traffic is now encrypted through the VPN
Your public IP should appear as the VPN server IP

Important Security Notes

  • Do not share your .conf file with anyone
  • Each .conf file is normally assigned to one device
  • If the VPN does not connect, turn it off and on again first
  • If it still fails, contact your VPN administrator
